# Aroha Sanctuary

A serene wellness website for inner balance and life transitions — built with React, Vite, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- **React** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **Radix UI** — accessible components
- **React Router** — routing
